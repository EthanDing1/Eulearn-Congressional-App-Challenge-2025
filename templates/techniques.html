Integration Techniques & Input Guide

Master integration methods and learn how to input expressions correctly

Input Instructions

Learn how to properly format your mathematical expressions for the integral solver.

Basic Operations

  • + Addition: x + 5
  • - Subtraction: x - 3
  • * Multiplication: 2*x or 2x
  • / Division: x/2 or (x+1)/(x-1)

Powers & Exponents

  • ^ or ** Power: x^2 or x**2
  • Square root: sqrt(x)
  • Cube root: cbrt(x)
  • General root: x^(1/n)

Trigonometric Functions

  • sin(x), cos(x), tan(x)
  • csc(x), sec(x), cot(x)
  • Inverse: asin(x), acos(x), atan(x)
  • Hyperbolic: sinh(x), cosh(x), tanh(x)

Logarithms & Exponentials

  • Natural log: ln(x)
  • Common log: log(x)
  • Exponential: e^x or exp(x)
  • Custom base: log(x, base)

Parentheses & Grouping

  • () Grouping: (x+1)*(x-1)
  • Nested: ((x+1)^2)/((x-1)^2)
  • With functions: sin(2*x + 1)
  • Always match opening and closing parentheses

Special Constants

  • e Euler's number: e
  • π Pi: pi
  • Examples: e^(pi*x)
  • Use constants directly in expressions

Integration Techniques

Essential integration methods for AP Calculus students.

Use if a composite function and its derivative are present in the integral.

Steps:

  1. Let \(u\) = inner function.
  2. Compute \(du\).
  3. Rewrite the integral in \(u\), integrate, then rewrite in terms of \(x\).

Example:

\[\int 2x \cos(x^2) \, dx\]

\(u = x^2\), \(du = 2x \, dx\)

\(\displaystyle \int \cos(u) \, du = \sin(u) + C = \sin(x^2) + C\)

Use when one function of the integral simplifies upon differentiation and the other can be easily integrated. Useful when methods like u-sub fail.

Formula:

\[\int u \, dv = uv - \int v \, du\]

LIATE rule:

The function in the integrand that appears first (left to right) in the LIATE order should be \(u\).

Log → Inverse Trig → Algebraic → Trig → Exponential

Example:

\[\int x e^x \, dx\]

\(u = x\), \(dv = e^x \, dx\)
\(du = dx\), \(v = e^x\)

Apply formula: \(\displaystyle \int u \, dv = uv - \int v \, du\)

\(\displaystyle \int x e^x \, dx = x \cdot e^x - \int e^x \, dx\)

\(= x \cdot e^x - e^x + C\)

\(= e^x(x - 1) + C\)

Eliminate radicals with trigonometric substitution.

Three cases:

  • For \(\sqrt{a^2 - x^2}\): substitute \(x = a\sin(\theta)\)

  • For \(\sqrt{a^2 + x^2}\): substitute \(x = a\tan(\theta)\)

  • For \(\sqrt{x^2 - a^2}\): substitute \(x = a\sec(\theta)\)

Steps:

Find appropriate case → substitute \(x\) → simplify radical → integrate to produce result in terms of \(\theta\) → use triangle (created by the substitution) to rewrite solution in terms of \(x\).

Example:

\[\int \sqrt{9 - x^2} \, dx\]

Pattern: \(\sqrt{a^2 - x^2}\) where \(a = 3\)

Substitute: \(x = 3\sin(\theta)\), \(dx = 3\cos(\theta) \, d\theta\)

\(\sqrt{9 - x^2} = \sqrt{9 - 9\sin^2(\theta)} = \sqrt{9\cos^2(\theta)} = 3\cos(\theta)\)

\(\displaystyle \int 3\cos(\theta) \cdot 3\cos(\theta) \, d\theta = 9 \int \cos^2(\theta) \, d\theta\)

\(\displaystyle = 9 \int \frac{1 + \cos(2\theta)}{2} \, d\theta\)

\(\displaystyle = \frac{9}{2}\left(\theta + \frac{\sin(2\theta)}{2}\right) + C\)

\(\displaystyle = \frac{9}{2}\left(\theta + \sin(\theta)\cos(\theta)\right) + C\)

Draw triangle: \(\sin(\theta) = \frac{x}{3}\), so \(\theta = \arcsin\left(\frac{x}{3}\right)\), \(\cos(\theta) = \frac{\sqrt{9-x^2}}{3}\)

Result: \(\displaystyle \frac{9}{2}\arcsin\left(\frac{x}{3}\right) + \frac{x\sqrt{9-x^2}}{2} + C\)

Breaking down a polynomial fraction into the sum of simpler fractions.

Steps:

  1. If numerator's degree ≥ denominator's degree → long division first.

  2. Factor denominator completely.

  3. Write the fraction as a sum of fractions, and solve for constants.

  4. Integrate each term.

Example:

\[\int \frac{3x+5}{(x+1)(x-2)} \, dx\]

Set up: \(\displaystyle \frac{3x+5}{(x+1)(x-2)} = \frac{A}{x+1} + \frac{B}{x-2}\)

Multiply both sides by \((x+1)(x-2)\):
\(3x + 5 = A(x-2) + B(x+1)\)

Solve for \(A\) and \(B\) by substituting smart values:

Let \(x = 2\): \(3(2) + 5 = 0 + B(3) \Rightarrow 11 = 3B \Rightarrow B = \frac{11}{3}\)

Let \(x = -1\): \(3(-1) + 5 = A(-3) + 0 \Rightarrow 2 = -3A \Rightarrow A = -\frac{2}{3}\)

\(\displaystyle \int \left[-\frac{2}{3} \cdot \frac{1}{x+1} + \frac{11}{3} \cdot \frac{1}{x-2}\right] dx\)

\(\displaystyle = -\frac{2}{3}\ln|x+1| + \frac{11}{3}\ln|x-2| + C\)

Integrals with trigonometric functions

Strategies:

  • \(\displaystyle \int \sin^m(x) \cdot \cos^n(x) \, dx\):


  • If one odd and one even: If sin is odd, save one sin. Write remaining \(\sin^2(x)\) as \(1-\cos^2(x)\) and substitute \(u=\cos(x)\). If cos is odd, save one cos. Write remaining \(\cos^2(x)\) as \(1-\sin^2(x)\) and substitute \(u=\sin(x)\).


  • If both odd: Pick either option from above.


  • If both even: use half angle or power reduction formulas.



  • \(\displaystyle \int \tan^m(x) \cdot \sec^n(x) \, dx\):


  • If sec is even: save a \(\sec^2(x)\) and use \(u=\tan(x)\). If tan is odd: save a \(\sec(x)\tan(x)\) and use \(u=\sec(x)\).


  • If neither condition is satisfied (such as tan even, sec odd) convert the integrand using the identity \(\sec^2(x) = 1 + \tan^2(x)\) and try to produce a \(\sec^2(x)\) factor or convert to sin/cos

Important Formulas:

  • \(\sin^2(x) + \cos^2(x) = 1\)

  • \(\sin(2x) = 2\sin(x)\cos(x)\)

  • \(\cos(2x) = \cos^2(x) - \sin^2(x) = 2\cos^2(x) - 1 = 1 - 2\sin^2(x)\)

  • \(1 + \tan^2(x) = \sec^2(x)\)

  • \(1 + \cot^2(x) = \csc^2(x)\)

  • \(\displaystyle \sin^2(x) = \frac{1 - \cos(2x)}{2}\)

  • \(\displaystyle \cos^2(x) = \frac{1 + \cos(2x)}{2}\)

Example:

\[\int \sin^3(x) \cdot \cos^2(x) \, dx\]

Strategy: sin is odd → save one \(\sin(x)\) and use \(u = \cos(x)\).

Rewrite \(\sin^3(x)\) as \((\sin^2(x))(\sin(x))\) and use \(\sin^2(x) = 1 - \cos^2(x)\):

\(\displaystyle \int \sin^3(x) \cdot \cos^2(x) \, dx = \int (1 - \cos^2(x)) \cdot \cos^2(x) \cdot (\sin(x) \, dx)\)

Let \(u = \cos(x) \Rightarrow du = -\sin(x) \, dx \Rightarrow \sin(x) \, dx = -du\)

Substitute:
\(\displaystyle -\int (1 - u^2) \cdot u^2 \, du = -\int (u^2 - u^4) \, du\)

Integrate:
\(\displaystyle -\left(\frac{u^3}{3} - \frac{u^5}{5}\right) + C\)

Replace \(u = \cos(x)\):
\(\displaystyle -\frac{\cos^3(x)}{3} + \frac{\cos^5(x)}{5} + C\)

When integrating products of exponential and trigonometric functions, integration by parts creates a cyclic pattern that allows us to solve algebraically.

Complete Example:

\[\int \sin(x) \cdot e^x \, dx\]

Step 1: First Application of Integration by Parts

Using the LIATE rule, choose:
\(u = \sin(x)\), \(dv = e^x \, dx\)
\(du = \cos(x) \, dx\), \(v = e^x\)

Apply the integration by parts formula:
\[\int u \, dv = uv - \int v \, du\]

\[\int \sin(x) \cdot e^x \, dx = \sin(x) \cdot e^x - \int e^x \cos(x) \, dx\]

Step 2: Second Application of Integration by Parts

We need to evaluate \(\displaystyle \int e^x \cos(x) \, dx\)

Choose:
\(u = \cos(x)\), \(dv = e^x \, dx\)
\(du = -\sin(x) \, dx\), \(v = e^x\)

Apply integration by parts again:
\[\int e^x \cos(x) \, dx = \cos(x) \cdot e^x - \int e^x \cdot (-\sin(x)) \, dx\]

\[\int e^x \cos(x) \, dx = e^x \cos(x) + \int e^x \sin(x) \, dx\]

Step 3: Substitute Back

Substitute this result into our equation from Step 1:
\[\int \sin(x) \cdot e^x \, dx = \sin(x) \cdot e^x - \left[e^x \cos(x) + \int e^x \sin(x) \, dx\right]\]

\[\int \sin(x) \cdot e^x \, dx = e^x \sin(x) - e^x \cos(x) - \int e^x \sin(x) \, dx\]

Step 4: Solve Algebraically

Notice that the original integral appears on both sides! Let \(I = \displaystyle \int e^x \sin(x) \, dx\)

\[I = e^x \sin(x) - e^x \cos(x) - I\]

Add \(I\) to both sides:
\[2I = e^x \sin(x) - e^x \cos(x)\]

Divide by 2:
\[I = \frac{e^x \sin(x) - e^x \cos(x)}{2}\]

Factor out \(e^x\):
\[I = \frac{e^x(\sin(x) - \cos(x))}{2}\]

Final Answer:

\[\boxed{\int \sin(x) \cdot e^x \, dx = \frac{e^x(\sin(x) - \cos(x))}{2} + C}\]

Key Insight:

The "cyclic" nature occurs because:
• Differentiating \(\sin(x)\) gives \(\cos(x)\)
• Differentiating \(\cos(x)\) gives \(-\sin(x)\) (returns to original function)
• Integrating \(e^x\) always gives \(e^x\) (stays the same)

This creates a loop that allows us to solve for the original integral algebraically!

Verification:

You can verify this answer by differentiating:
\[\frac{d}{dx}\left[\frac{e^x(\sin(x) - \cos(x))}{2}\right] = \sin(x) \cdot e^x \,\checkmark\]
